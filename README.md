# Amaretto Labs

Small-batch open-weight language models, tuned for literary writing and sized for the hardware people actually have.

**Website:** [amaretto.ai](https://amaretto.ai)

## Models

| Model | Parameters | Context | License | Weights |
|-------|-----------|---------|---------|---------|
| Amaretto-3B | 4 B | 256 K | Apache-2.0 | [Hugging Face](https://huggingface.co/AmarettoLabs/Amaretto-3B) |
| Amaretto-8B | 9 B | 256 K | Apache-2.0 | [Hugging Face](https://huggingface.co/AmarettoLabs/Amaretto-8B) |
| Amaretto-14B | 15 B | 256 K | Apache-2.0 | [Hugging Face](https://huggingface.co/AmarettoLabs/Amaretto-14B) |

All models ship with GGUF quantizations and run locally with llama.cpp, Ollama, and friends.
